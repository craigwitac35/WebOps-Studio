Featured Project

Guardian 4 Heroes

Guardian 4 Heroes is an equine therapy nonprofit serving veterans and first responders — people I have a deep connection with as a veteran myself. They needed a site that felt emotionally grounded, mission-driven, and easy to navigate. Built entirely from scratch in HTML, CSS, and vanilla JavaScript. Lightweight, accessible, and conversion-focused throughout.

The build required careful attention to emotional tone — the site had to communicate trust and mission before it asked for anything. Every layout decision was made with that goal in mind: from the full-viewport hero image to the tiered donation buttons to the mobile-first navigation structure.

05 · Code
View Fullscreen
Code Breakdown · 05

styles.css: Under the Hood

Organized sections, a CSS custom property token system at the top, and clear comment headers throughout. Not just a working stylesheet — a maintainable one.

L4-18

:root token system — all colors, spacing, and shadows as custom properties. Change one value, update the entire site.

L20-22

Section comment headers — every section labeled and separated. Any developer knows exactly where everything lives.

L49-56

body::before pseudo-element — site-wide overlay effect handled in pure CSS. Zero JavaScript required.

L38-42

background-attachment: fixed — parallax scroll is a single CSS property. No libraries, no JavaScript.
