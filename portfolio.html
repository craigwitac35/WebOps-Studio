Guardian 4 Heroes

HTML CSS Vanilla JS GitHub Pages Equine Therapy
View Live Site →

The Problem

Guardian 4 Heroes is an equine therapy nonprofit serving veterans. They had no web presence — just a mission and a name. They needed a site that would communicate the emotional weight of their work while remaining clean and trustworthy enough to attract donors and program participants.

The Result

A fully custom site built and deployed on GitHub Pages. Photography-forward layout, clear program information, and a donation pathway. The client reports consistent compliments and increased program inquiries since launch.

"Craig went above and beyond. He didn't just build us a website — he brought our mission to life. We get compliments on it constantly."
