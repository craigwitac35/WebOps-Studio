Featured Project

Mission Possible: Hooves & Paws Mud Run

A complete event registration platform for a Minnesota nonprofit fundraiser benefiting equine therapy and K9 training programs for veterans and first responders.

The project was scoped to be more than a marketing site — the organizers needed a full system to manage registrations, track payments, and handle the day-to-day work of running a real event. Everything was built from scratch, including the design system and the admin tools.

What was built: a cinematic homepage with a hand-composited logo; an events page with day-of details and pricing; a multi-participant registration form with t-shirt sizing and dynamic pricing rules; unique reference codes tied to Givebutter payment integration; a confirmation page with copy-to-clipboard reference codes; and a private admin dashboard with live stats, search, and shirt-order reporting for the event hosts.

The site went live on a custom domain via Vercel with full DNS configuration through IONOS.

Nonprofit Site

Guardian 4 Heroes

A multi-page website for a Minnesota nonprofit offering equine-facilitated therapy and training to veterans and first responders. The site needed to feel warm and personal while still being a clear front door for programs, donations, and upcoming events.

What was built: a full multi-page site with home, about, programs, shop, donate, and contact pages; an announcements section the team can update for time-sensitive news; and a feature module that highlights the Mission Possible mud run with a direct registration link.

The site is hand-coded HTML/CSS/JS and hosted on GitHub Pages with a custom domain.
