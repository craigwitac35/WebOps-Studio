FAQ

Common Questions

Do you use templates or WordPress?

No. Everything I build is coded from scratch using HTML, CSS, and JavaScript — fully custom, always.

Will my site work on mobile?

Yes. Every project is built mobile-first and tested on all screen sizes before delivery.

Can I request changes after v1?

Absolutely. Every project includes 1–2 rounds of revision so we get it right together.

Do you connect forms or donation tools?

Yes — contact forms, PayPal/Stripe links, and basic integrations are included where needed.

How much do you charge?

Every project is different. I'll work with you to create a fair quote based on your needs and budget.
