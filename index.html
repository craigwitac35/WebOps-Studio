Mission Possible: Hooves & Paws

A custom event registration platform for a Minnesota nonprofit fundraiser benefiting equine therapy and K9 training programs for veterans and first responders.

The organizers needed more than a marketing site – they needed a full system to manage registrations, track payments, and run a real event. Everything built from scratch.

Next.js TypeScript Supabase Vercel Givebutter
  • Custom design system — cinematic homepage, hand-composited logo, full mobile-responsive layout.
  • Registration flow — multi-participant forms, dynamic pricing, t-shirt sizing.
  • Payment tracking — unique reference codes, partial-payment handling, Givebutter integration.
  • Admin dashboard — live stats, registration search, shirt order reporting.
